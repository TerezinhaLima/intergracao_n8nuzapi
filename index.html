Integração WhatsApp + n8n

Documentação técnica para envio de mensagens via API REST

1. Configuração Uzapi

Primeiro, realize o cadastro na Uzapi e configure sua instância no painel:

  • Session: Identificador único (ex: empresa1).
  • SessionKey: Chave de segurança (ex: empresa1).
  • Token: Chave mestre fornecida via e-mail/painel.

2. Parâmetros n8n (HTTP Request)

Endpoint

Method: POST

URL: SUA_URL_UZAPI/sendText

Header Auth (Generic)

Name Value
apitoken Sua Session do Endpoint
sessionkey Sua SessionKey do Endpoint

Body Content (JSON)

{
  "session": "sua_session",
  "number": "5511999999999",
  "text": "{{ JSON.stringify($json.mensagem) }}"
}
